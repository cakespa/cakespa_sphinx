ようこそ
========

このサイトは、CakePHPのユーザの皆さんが集まって、温泉に入りながらCakePHP話で盛り上がる恒例イベント「CakePHP温泉部」の活動報告サイトです。
温泉部の活動は大変にゆるく、のんびり和やかに行われますが、その際に有用なTipsなどについて議論される場合もあります。また、活動を通して一つの成果物がまとめられる事もあります。

これら有用な情報を他の皆さんにも共有できるよう、アーカイブとしてまとめました。
ここにまとめられたドキュメントは全て、CakePHP温泉部で共有されたものです。

CakePHP温泉部とは
=================

CakePHPの情報共有のために作られたイベントです。
主催者は `ECWorks <http://blog.ecworks.jp/>`_ の `MASA-P <http://twitter.com/ecworks_masap>`_ で、運営スタッフとして `1-byte.jp <http://1-byte.jp/>`_ の `tfmagician <http://twitter.com/tfmagician>`_ が参加しています。
情報共有は実際建前で、 *温泉に入って日頃の疲れを癒す* ことを目的としています。日頃の激務の中では、せっかく思いついたユニークなアイデアもかき消されてしまいがちですが、のんびりと過ごす中で何らかのインスピレーションを感じる事もあるかと思います。

CakePHPのイベントはほとんどがカンファレンススタイル・あるいはハッカソンスタイルのものが多いですが、温泉部ではこのような形式にとらわれていません。温泉という縛りこそありますが、この癒しの時をのんびりと過ごし、その中で同じくCakePHPを使われている皆さんとの交流を大事にしていただきたいと考えています。

現在毎年1回のペースで開催されており、今回は第2回として熱海で実施される予定です。

* サイト `http://www.cakespa.org/ <http://www.cakespa.org/>`_ (このサイト)
* 連絡先 info [at] cakespa.org
* Twitterアカウント `@cakespa <http://twitter.com/cakespa>`_
* Twitterハッシュタグ `#cakespa <https://twitter.com/search?q=%23cakespa>`_

CakePHP温泉部への貢献
=====================

このドキュメントは `github <http://github.com/>`_ で管理されています。
このページに追加のTipsを加えたい、あるいは修正したい方は以下のリポジトリをフォークして下さい。

* `cakespa_sphinx <https://github.com/cakespa/cakespa_sphinx>`_

ドキュメントの書き方やビルドの方法は、リポジトリのREADMEに含まれています。

.. note::
   **Forkしたリポジトリを最新にする方法**

   Forkしたリポジトリは、修正を加える前に元リポジトリの最新コミットをpullしてからにしましょう。
   
      | 【リモートリポジトリを登録しておく(１回のみ実行)】
      | git remote add -f cakespa git://github.com/cakespa/cakespa_sphinx.git
      | 【フォークしたリポジトリを最新版にする】
      | git pull cakespa master
      | git push origin master
